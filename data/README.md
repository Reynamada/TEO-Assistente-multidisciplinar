# Dados do TEO

Este diretório contém dados estáticos e de referência para o sistema TEO.

## Arquivos

| Arquivo | Descrição |
|---------|-----------|
| `tipos_sessao.json` | Lista de tipos de sessão disponíveis na clínica |
| `glossario_clinico.json` | Termos técnicos e suas traduções para linguagem dos pais |

> [!IMPORTANT]
> Dados de pacientes reais NUNCA devem ser armazenados aqui.
> Todos os dados clínicos ficam exclusivamente no PostgreSQL.
